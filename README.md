# Image Segmentation using k-means cluster of OpenCV  ROS 2

### Requirement
```
Ubuntu 20.04
ROS 2 Foxy
Python 3.8
Setuptools 
OpenCV-Pythons

A webcam
```

### Clone the package and build the node
```
git clone

cd /path/to/workspace/src


cd /path/to/workspace
colcon build --packages-select ros2foxy_opencv
```
